# ❤️ 나는 솔로 콘셉트 포트폴리오

"나는 솔로"에 진심인 제가, 이를 콘셉트로 제작한 웹 기반 포트폴리오를 소개합니다.
프로그램처럼 저를 하나의 출연자처럼 풀어내었고, 기술 스택은 족자 스타일을 활용해 독특하게 디자인했습니다.
또한, 지금까지 진행한 프로젝트들을 짧고 임팩트 있게 정리하여 한눈에 보기 쉽게 구성했습니다.

## 🛠️ 사용 기술

-   React
-   Tailwind CSS
-   Vite
-   JavaScript
-   Vercel (배포)

## 🌟 주요 특징

-   "나는 솔로" 느낌을 살린 개성 있는 디자인
-   기술 스택은 족자 형태로 구현
-   프로젝트는 스와이퍼로 부드럽게 넘기기
-   반응형 웹 지원
